CppMem: Interactive C/C++ memory model exploration tool

CppMem can either be used from the command-line or as a web
application, with the main code running as JavaScript in the browser
and a web server for graph layout. 

Documentation:
- for the context: http://www.cl.cam.ac.uk/~pes20/cpp/
- for installation instructions: INSTALL.txt
- for using the tool: (local) web/help.html
